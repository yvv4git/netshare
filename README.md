# NetShare
Share files by http and may be other protocols.


## HOW TO
Running the program if the config is located in the user's directory.  
```
go run main.go
go run main.go -p 8181
go run main.go -s 127.0.0.1
go run main.go -p 8181 -s 127.0.0.1
```
